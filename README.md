# NirnWarden

**NirnWarden** је Paper Minecraft додатак за верзију 1.21.11 дизајниран за *Сезону 3* приватног сервера *Нирн*. Администраторима сервера пружи потпуну контролу над понашањем мобова који оштећују блокове.

[![Test and Release](https://github.com/crnobog69/nirnwarden/actions/workflows/main.yml/badge.svg)](https://github.com/crnobog69/nirnwarden/actions/workflows/main.yml)

## Могућности
Спречите следеће активности мобова:
- Експлозије Creeper-а
- Експлозије Ender Crystal-а
- Уништавање блокова од стране Ender Dragon-а
- Подизање/постављање блокова од стране Enderman-а
- Ватрене куглe Ghast-а
- Једење усева од стране зечева
- Једење траве од стране оваца
- Инфестацију блокова од стране Silverfish-а
- Бербу усева од стране сељана
- Експлозије при стварању Wither-а
- Пројектиле Wither лобање

## Команде
Све команде захтевају дозволу `nirnwarden.admin` (подразумевано: OP).

| Команда | Алијас | Опис |
| --- | --- | --- |
| `/nw help` | `/nirnwarden help` | Прикажи доступне команде |
| `/nw status` | `/nirnwarden status` | Прегледај тренутна подешавања заштите |
| `/nw set <опција> <true/false>` | `/nirnwarden set ...` | Укључи/искључи одређену опцију |
| `/nw reload` | `/nirnwarden reload` | Поново учитај конфигурацију из фајла |

## Конфигурација
Фајл `config.yml` омогућава вам да укључите/искључите заштите.
`true` = Оштећивање је **БЛОКИРАНО** (Заштита укључена)
`false` = Оштећивање је **ДОЗВОЉЕНО** (Заштита искључена)
```yaml
creeper_griefing: true
enderman_griefing: true
wither_griefing: true
# ...
```

## Како компајлирати

Да бисте компајлирали овај додатак из изворног кода, потребан вам је инсталиран JDK 21.

1. Отворите терминал у фолдеру пројекта.
2. Покрените команду за компајлирање:
   - **Windows**: `.\gradlew.bat shadowJar`
   - **Linux/Mac**: `./gradlew shadowJar`

### Где се налази JAR фајл?
Када се компајлирање успешно заврши, можете пронаћи компајлирани JAR фајл додатка у:
`build/libs/`

Фајл ће имати назив попут `NirnWarden-YYMMDD-HHMM-SNAPSHOT.jar`.
